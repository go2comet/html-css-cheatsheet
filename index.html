My HTML & CSS Cheatsheet

HTML

Section Review Resource Link
Elements HTML stands for HyperText Markup Language and is used to create the structure and content of a webpage. mdn "HTML: HyperText Markup Language"
Elements Any visible content should be placed within the opening and closing <body> tags. mdn "<body>: The Document Body element"
Elements <p>, <span> and <div> tags specify text or blocks. geeksforgeeks "Difference between <div> and <span> Tag in HTML
Elements Images (<img>) and videos (<video>) can be added by linking to an existing source.
Structure Information about the web page, like the title, belongs within the <head> of the page. mdn "<head>: The Document Metadata (Header) element
Structure Anchor tags (<a>) are used to link to internal pages, external pages or content on the same page. mdn "<a>: The Anchor element
Structure Comments are written in HTML using the following syntax: <!-- comment -->.
Semantic Semantic HTML is a modern standard and makes a website accessible for people who use screen readers to translate the webpage and improves your website’s SEO. W3 Schools: "HTML Semantic Elements"
Semantic <header>, <nav> , <main> and <footer> create the basic structure of the webpage. mdn "Semantics in HTML"

CSS

Section Review Resource Link
Setup and Syntax An internal stylesheet is written using the <style> element inside the <head> element of an HTML file. mdn "CSSStyleSheet"
Setup and Syntax External stylesheets are linked to HTML using the <link> element.
Selectors IDs are more specific than classes, and classes are more specific than type. W3 Schools "CSS Specificity"
Visual Rules Text can have two different color attributes: color and background-color. color defines the color of the text, while background-color defines the color behind the text.
Visual Rules The !important flag will override any style, however it should almost never be used, as it is extremely difficult to override. W3 Schools "CSS The !important Rule"
